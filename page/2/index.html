数组 Array

  • 定义数组的格式：var <varName> [n]<type,n> = 0

从排序数组中删除重复项

发表于 2020-01-05 | 阅读次数:
字数统计: 488 | 阅读时长 ≈ 2

题目地址

题目地址：https://leetcode.com/problems/remove-duplicates-from-sorted-array/

思路

  • 数组下标
  • 快慢指针

使用快慢指针来记录遍历的坐标。

  • 开始时这两个指针都指向第一个数字
  • 如果两个指针指的数字相同，则快指针向前走一步
  • 如果不同，则两个指针都向前走一步（之后将慢指针所在的节点值换成快指针所指位置）
  • 当快指针走完整个数组后，慢指针当前的坐标加1就是数组中不同数字的个数
