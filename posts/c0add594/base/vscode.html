2. Eslint

注意：同时使用Vetur与ESlint在代码格式检查方面存在冲突，存在以下场景：

Vetur使用快捷键Alt+Shift+F时，会将字符串由双引号包裹起来"string"，如果配置了ESlint的自动保存，在使用快捷键Alt+S时，会将字符串用单引号包裹起来'string'，等等其他的问题

为了解决这个问题，可以在setting.json中进行调整：

"vetur.format.defaultFormatter.js": "vscode-typescript",
"editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
}
